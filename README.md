# Portfolio Website - v2.0

Mon portfolio personnel développé avec **Next.js** et **TypeScript** pour présenter mes projets web, UX/UI et motion design.

🌐 **Live Demo** : [https://noahbucheton.fr/](https://noahbucheton.fr/)

## TL;DR

Tu peux t’inspirer de ce repo pour créer ton propre portfolio.  
Si tu le réutilises, merci de me créditer en mentionnant ce repo ou mon profil GitHub 🙌

---

## Built With

Ce projet a été réalisé avec les technologies suivantes :

- **Next.js 14** (App Router)
- **React**
- **TypeScript**
- **Tailwind CSS**
- **Static Export** (`output: 'export'`)
- **Infomaniak** (déploiement FTP)

---

## Features

- ✅ Portfolio responsive (desktop / mobile)
- ✅ Design orienté **UI tech / terminal**
- ✅ Pages projets dynamiques exportées en statique (`/projects/[id]`)
- ✅ Études de cas par projet (contexte, objectifs, stack, captures, vidéos)
- ✅ Sections : Projets / Compétences / Parcours / À propos / Contact
- ✅ Compatible hébergement classique (Apache / FTP)
